## 📈 이야기 흐름 (왜 이 순서인가)
1. **셋업(00)** 환경·전문가 검증 → **기본 보행(01)** 평지 87%.
2. **미로(02·03·04)** 8%→43%→58~65%에서 정체 — "턴 약점/기하 한계"로 진단했었음.
3. **그런데** 영상에서 개미가 **뒤집혀 기어가는** 걸 발견 → `up_z` 진단(미로 워커는 60% 스텝 전복).
4. **자세 종료 + 직립 보너스(05·06)** → 평지 87→98%, 미로 58→88%. 턴 약점의 큰 부분이 **전복의 결과**였음을 입증.
5. **풀맵 검증(07)** 축소기둥 없는 진짜 풀맵서도 fine-tune 73% → 70% 게이트 정직 통과(축소기둥은 디딤돌이었음).
